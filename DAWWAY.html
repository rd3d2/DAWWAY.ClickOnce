FAQ

Which DAWs and control surfaces does it support?


DAWWAY Lite is designed for wide compatibility across a wide range of DAWs and control surfaces:

DAWS
Ableton Live, BITWIG, Cubase / Nuendo, Luna, FL Studio, Mixcraft, Reaper, Reason, Studio One.

CONTROL SURFACES
 AVID - Artist Mix, AVID - Artist Control, Behringer - BCF2000, Behringer - X-TOUCH, Behringer - X-TOUCH COMPACT, Behringer - X-TOUCH MINI, Behringer - X-TOUCH ONE, Behringer - Wing, Emagic - Logic Control, Euphonix - MC Mix, Euphonix MC Control, Euphonix - System 5, Frontier Design - AlphaTrack, iCON - Platform M+, iCON - Platform X+, iCON - QCon Pro, iCON - QCon Pro G2, iCON - QCon Pro X, iCON - QCon Pro XS, iCON - P1-Nano, iCON - V1-M, iCON - V1-X, Korg - nanoKONTROL2, M-Audio - ProjectMix I/O, Mackie d8b (emulation), Mackie - MCU Pro, Mackie - MCU XT Pro, Mackie - Mackie Control, Mackie - Mackie Control Universal, Mackie - Mackie Control Universal Pro, PreSonus - FaderPort, PreSonus - FaderPort 8, PreSonus - FaderPort 16, Roland - VS700c, Solid State Logic - Nucleus, Solid State Logic - Nucleus 2, Solid State Logic - UF1, Solid State Logic - UF8, Steinberg - CMC, Tascam - FW-1082, Tascam - FW-1884, Tascam - US-2400, Yamaha - 01X

Do I need an extra monitor?

No — but it shines with an external display at your eyeline (including touchscreens).

Is it difficult to set up?

The goal is “setup once, always there.” Configure it, then launch and go.

What do I get with Pro?

Pro adds track text, analogue meters,new sysex specifications (Long Track Names, Buttonised Markers, Automated Track Numbering), and extended theming.
